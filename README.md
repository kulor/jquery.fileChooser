jquery.fileChooser
==================

jQuery plugin to add some behaviours to a custom designed file chooser

Usage:
------

    $('.file-chooser').fileChooser({
        fileSelector: 'input[type="file"]',
        fileLabelSelector: '.file-paths',
        helpBlockSelector: '.help-block'
    });
